# Google Places Lead Scraper (n8n)

Automated lead generation workflow that extracts verified local business data for outreach campaigns.

## Features

- Scrapes business listings using Google Places API
- Extracts business name, phone number, address, and website
- Deduplicates listings
- Separates businesses with websites and without websites
- Saves leads to Google Sheets

## Use Case

Agencies and sales teams can generate hundreds of clean business leads in minutes for cold outreach.

## Technologies

- n8n
- Google Places API
- Google Sheets
- HTTP Requests
